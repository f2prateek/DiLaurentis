# DiLaurentis
Pretty print JSON documents.

### Installation
`go get github.com/f2prateek/DiLaurentis`

### Usage
```
dilaurentis [--indentation indent]
dilaurentis -h | --help
dilaurentis --version

Options:
  --indentation indent    Indentation [default:   ]
  -h --help               Show this screen.
  --version               Show version.
```
